# Kudurru Kings — Colyseus Starter (Rebuilt)

This is a minimal **server + client** starter to get you playing quickly with Colyseus (TypeScript) and a Vite web client.

## Quick start
### 1) Server
```bash
cd server
npm i
npm run dev   # ws://localhost:2567
```
Monitor is available at `http://localhost:2567/colyseus`.

### 2) Client
```bash
cd client
npm i
npm run dev   # http://localhost:5173
```
