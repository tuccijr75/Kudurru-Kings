Kudurru Kings — Client UI Fix
----------------------------------
This zip contains two files to drop into your Vite client:

  client/src/main.ts
  client/src/style.css

Steps:
1) In your project, stop Vite if running (Ctrl+C in the client terminal).
2) Replace the two files above with the ones from this zip.
3) Restart Vite:  npm run dev
4) Open http://localhost:5173 → Join Room.

Changes:
- Compact cards (50% size), flip-for-details, tooltips, legend, zoom slider.
- Reveal AI enabled on join (for testing).
